## 스크립트 작성법 

매번 환경설정을 쉘 별로 일일히 하지 말고 스크립트로 자동화를 하도록 하자. 

vim [스크립트이름].sh

```
#!bin/bash

source /opt/ros/humble/setup.bash

sleep 1

set -o errexit

set -o nounset

set -o pipefail

exit

```

sudo chmod +x [스크립트이름].sh

./[스크립트이름].sh


더 자세한 사항은 bash style guild를 참고하거나 gemini에게 물어보자. 


## vim 사용법 

```
vi .vimrc  # 여기서 vim 관련 세팅할 수 있음. 꾸미기같은거 

sudo cp ~/.vimrc /root/.vimrc

vi s.sh
sudo chmod +x s.sh

./s.sh

```

vim 설정을 미리 해둬야 정신건강에 좋으니 빨리 세팅 해두자.
