**adhere**
: 들러붙다, 부착되다

**agnostic**
: 불가지론자, 어떤 운영체제나 프로세서의 조합에 대한 지식 없이도 기능을 수행할 수 있는 소프트웨어 기술

**arguably**
: 틀림없이

**arbitrary**
: 임의적인, 제멋대로인

**as alluded to**
: 언급했듯이, 암시했듯이

**assert**
: (사실임을 강하게) 주장하다

**attempt to**
: ~하려고 시도하다

**bespoke**
: (개인 주문에 따라) 맞춘, 맞춤 생산을 하는

**compliance**
: 준수, 따름

**comprehensive**
: 포괄적인

**comprise**
: ~으로 구성되다

**concern**
: (사람에게) 영향을 미치다, (무엇에) 관한 것이다, 우려

**constrain**
: ~하게 만들다

**consistent**
: 한결같은, 일관된

**consistently**
: 일관하여, 지속적으로

**demonstrate**
: 증거를 들어가며 보여주다, 입증하다, (행동으로) 보여주다, 시위에 참여하다

**diverse**
: 다양한

**eliminate**
: 없애다, 제거하다

**encompass**
: (많은 것을) 포함하다

**equalizer**
: 동점골

**expose**
: 드러내다

**feasible**
: 실현 가능한

**fuse**
: 융합되다

**govern**
: 통치하다

**hamper**
: 방해하다

**idiomatic**
: 자연스러운, (언어 표현이 모국어 사용자가 쓰는 것 같이) 자연스러운

**impede**
: 지연시키다

**impose**
: 도입하다, 부과하다, 강제하다

**inability**
: 무능, 불능

**infrastructure**
: 사회 기반 시설

**inherent**
: 내제하는

**integration**
: 통합

**interrelate**
: 밀접한 연관을 갖다

**introspection**
: 내성, 자기 성찰

**in turn**
: 차례처례, 결국

**legacy**
: 유산

**leverage**
: 영향력, 지렛대 사용

**lossy**
: (파일 정보가) 손실되는

**marshal**
: 진행요원, 모으다, 통제하다

**mass**
: 덩어리, (양이) 많은

**mirror**
: 잘 보여주다, 반영하다, 비추다

**misuse**
: 남용하다

**myraid**
: 무수한

**notable**
: 주목할 만한

**notably**
: 특히, 현저히

**nurture**
: (잘 자라도록) 양육하다

**obligation**
: 의무

**pars**
: 부분, 구조의 특정 부분

**pervasive**
: 만연하는

**plumbing**
: (건물의) 배관, 배관 작업

**popularize**
: 많은 사람들에게 알리다, 대중화하다, 보급하다

**prevalent**
: 일반적인

**rationale**
: (특정한 결정, 행동 방침, 신조 등의) 이유

**readiness**
: 준비가 되어 있음

**refer to somebody/something**
: ~을 나타내다

**refer to**
: ~을 나타내다

**resilience**
: (충격, 부상 등에서의) 회복력

**rigorous**
: 철저한, 엄격한

**rover**
: 방랑자

**satellite**
: 위성

**scarce**
: 부족한, 드문

**showcase**
: 공개행사

**sophisticated**
: 세련된, 교양 있는

**sophisticate**
: 세련된(교양있는) 사람

**state of the art**
: 최첨단의, 최신식의

**stem from**
: ~에 기인하다

**struggle**
: 투쟁하다, 싸우다, 힘겹게 나아가다

**tailor**
: 재단사

**tailored**
: 잘 맞도록 만든

**terrain**
: 지형, 지역

**warehouse**
: 창고
