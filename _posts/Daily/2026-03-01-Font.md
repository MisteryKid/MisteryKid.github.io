## vim 세팅

vim을 아무 설정없이 그냥 사용하면 너무 스트레스 받음

아래 내용은 단순 설치가 아닌 사용자가 더 편리하게 툴을 사용하기 위한 config 설정 방법임.

```shell
vim ~/.vimrc
```

vim이 열리면 아래 내용 입력
```Vimscript
" --- 1. 기본 가독성 및 인터페이스 ---
set number              " 왼쪽에 줄 번호 표시
set cursorline          " 현재 커서 줄 강조 (터미널 배경과 대비되어 예뻐짐)
set showmatch           " 괄호 짝 찾기 기능
syntax on               " 코드 문법 강조 활성화

" --- 2. 들여쓰기 (코딩 표준 세팅) ---
set autoindent          " 자동 들여쓰기
set tabstop=4           " Tab을 4칸으로 설정
set shiftwidth=4        " 자동 들여쓰기 시 4칸 이동
set expandtab           " Tab을 공백(Space)으로 변환 (협업 필수)

" --- 3. 색상 및 그래픽 (지금 터미널에 필수!) ---
set termguicolors       " 터미널의 1600만 가지 색상(True Color)을 그대로 사용
set background=dark     " 어두운 배경 모드 최적화

" --- 4. 편의 기능 ---
set mouse=a             " 마우스 클릭, 드래그, 스크롤 허용 (Vim이 편해짐)
set clipboard=unnamedplus " 시스템 복사/붙여넣기 공유 (설정에 따라 안될 수 있음)
set ignorecase          " 검색 시 대소문자 무시
set smartcase           " 검색어에 대문자가 있으면 대소문자 구분

" --- 5. 파스텔톤 수동 색상 포인트 (선택 사항) ---
" 터미널 색상과 별개로 특정 부분만 더 튀게 하고 싶다면 사용하세요.
hi Comment ctermfg=gray guifg=#9399b2  " 주석 색상을 더 부드러운 파스텔 회색으로
hi String ctermfg=green guifg=#a6e3a1   " 문자열 색상을 캣푸친 그린으로
```

이 외에 더 많은 세팅이 있으니 필요하면 검색해보는것도 나쁘지 않다.
