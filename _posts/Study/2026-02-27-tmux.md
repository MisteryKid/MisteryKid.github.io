# 설치 방법
```
sudo apt update
sudo apt install tmux
```


# 사용법

``` shell
# 시작하기
tmux

# 화면 가로로 쪼개기 
Ctrl + b + \" 

# 화면 세로로 쪼개기
Ctrl + b + % 

# 창 사이 이동하기
Ctrl + b를 누른 상태에서 손을 떼고 방향키 입력.

# 세션에서 잠시 나오기 (종료 아님!): 
Ctrl + b 누른 후 d (Detach)

# 세션 종료
exit

# 세션 리스트 보기: 
tmux ls

```

# 기존 명령어
```shell

기능	단축키	설명
가로 분할	Prefix + -	위/아래로 화면을 나눕니다.
세로 분할	Prefix + |	좌/우로 화면을 나눕니다.
전체 화면	Prefix + z	현재 칸을 최대화/이전 크기로 복구합니다 (Zoom).
패널 닫기	Prefix + x	현재 칸을 닫습니다 (하단에 y/n 확인 창 뜸).
패널 이동	Prefix + 방향키	상/하/좌/우 패널로 커서를 옮깁니다

새 창 생성: Prefix + c

다음 창 이동: Prefix + n (Next)

이전 창 이동: Prefix + p (Previous)

창 이름 변경: Prefix + ,


세션 목록: Prefix + s (다른 세션으로 즉시 이동 가능)

세션 탈출: Prefix + d (Detach - 작업 상태 유지하며 터미널 밖으로 나가기)

세션 재접속: 터미널에서 ta (혹은 tmux a) 입력


설정 새로고침: Prefix + r (우리가 만든 .tmux.conf 반영용)

복사 모드: Prefix + [ (방향키로 위로 올라가서 지난 로그를 볼 수 있음. 나오려면 q)


```



# conf 설정

``` shell


# conf파일 생성 
vim ~/.tmux.conf


# 쉘에 적어야하는거 
# 1. Alt + h 또는 v로 즉시 분할 (Prefix 필요 없음!)
bind -n M-h split-window -h   # 세로 분할 (좌우)
bind -n M-v split-window -v   # 가로 분할 (상하)

# 2. Alt + 화살표 키로 창(Pane) 사이 이동
bind -n M-Left select-pane -L
bind -n M-Right select-pane -R
bind -n M-Up select-pane -U
bind -n M-Down select-pane -D

# 3. F2 키로 새 창(Window) 만들기
bind -n F2 new-window

# 4. F3, F4 키로 이전/다음 창 이동
bind -n F3 previous-window
bind -n F4 next-window


# 마지막에 소싱해야 적용됨 
tmux source-file ~/.tmux.conf



tmux kill-server
```
