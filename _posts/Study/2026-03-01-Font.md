개발자들이 자주 사용하는 폰트가 있다는 말을 듣고 세팅 시작. 

D2coding이 한글 폰트도 지원해서 사람들이 많이 쓴다고 함

+ tmux의 theme 설정할 때 nerd font를 미리 깔아두라 해서 설치 진행 중



1. 아래의 사이트에서 D2 coding nerd font 다운
https://www.nerdfonts.com/font-downloads 

![폰트](/assets/2026-03-01/D2Coding_nerd_font.png)


2. zip 파일로 된거 압축 해제
![alt text](/assets/2026-03-01/after_decomp.png)
압축되면 여러 폰트가 뜨는데 거기서 마음에 드는거 install

필자는 D2CodingLigatureNerdFont-Regular.ttf 이거 택함

![alt text](/assets/2026-03-01/installfont.png)

3. reboot 후 터미널에 적용

터미널 -> preference -> profile -> custom font -> D2coding 설정

![alt text](/assets/2026-03-01/set_profile.png)

폰트가 적용되면 다음과 같이 변한다. 
![alt text](/assets/2026-03-01/applyfont.png)
