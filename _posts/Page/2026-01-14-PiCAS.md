Q: How should a user assign the priorities to the callbacks?
 Are any rules of thumbs or like a shorther task get a highter priority?

A: We propose the priority assignment scheme. So that assignment scheme is trying to satisfy our callback scheduling strategy.
So we propose the private assignment scheme so if you are interested in on how to assign the callback priority, maybe you can list our paper and see the algorithm 


Q : Do these priorities have to be unique or can i habe callbacks with the same priority?

A : that is a good question. yes exactly right. So we need a required to assign a unique priority for all the callbacks


Q: Would it be possible to also have a pipelining for a chain or can chain only be executed at a time?

A : Based on how the locatoin, algorithm... it's not always allocated on one chain to one executor.
It depends on the utilization of cpu and some other parameters like the when we allocate the node to executor actually, it should satisfy the scheduling strategy.
